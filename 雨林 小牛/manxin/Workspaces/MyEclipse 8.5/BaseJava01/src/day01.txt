JAVA:
	JDK:Java开发的工具包，进行Java开发所需的基本类库和JRE
	JRE:Java程序的运行环境，JRE包括JVM
	JVM:Java的虚拟机，Java通过虚拟机实现的跨平台
	
	特点：跨平台，内存的管理(垃圾回收机制)，强制类型(变量必须定义类型后才能使用)
	
Java的基本数据类型
	逻辑值型：
		boolean:true or false
	数值型:
		整数类型：
			byte: 8位2进制大小   -127~128
			short：16位2进制大小   -32767~32768
			char: 16位2进制大小    0~65535
			int：32位2进制大小	
			long：64位2进制大小
			
		小数类型：	
			float：单精度浮点型
			double：双精度浮点型
						
	自动类型转换：
	强制类型转换：利用小括号，强制的声明一个数据的类型
	
注释：
	软件：源代码 + 注释 + 可执行的文件 + 软件说明文档
	单行注释：//
	多行注释：/* */   /**  */
	文档注释：帮助文档
	特殊的注释：//TODO 	//XXX
	
程序的结构：
	顺序结构：
	选择结构：if/else	switch/case	goto(保留关键字)
	循环结构：for,while,do while
	
变量：
	变量是一小块儿的内存空间
	声明/定义：给变量起名并且指定数据类型	int i
	赋值：往变量中存数据	i=1
	初始化：声明 + 赋值 int i =1   缺省 = 默认
	
	通过操作变量的标示符(变量名)来操作变量中的数据
	变量只有初始化之后才能使用
	
	变量的作用域：
		变量的使用范围
		变量在声明时，所在的{}内
		在同一作用域内，同一个变量名不能够声明多次
		
运算符：
	算术运算符：
		+	-	*	/	%: 17%5=2
	自运算符：
		++	--	+=	-=	*=	/=
		int i=0;
		i++和++i:++在前，先自增1，++在后，后自增1
	关系运算符：
		>	<	==	!=	>=	<=
	逻辑运算符：
		&&	||	!	&	|
	条件运算：三目运算符
		逻辑表达式  ? 结果1 : 结果2
		
选择结构:
	switch/case
	switch(int temp){
		case 1:代码1;break;
		case 2:代码2;break;
		default:代码3;
	}
	解释：根据temp的值，执行代码1或2，如果temp的值不为其中任何，则执行代码3，temp也可以是字符
	
	if/else if/else		如果/再如果/否则
